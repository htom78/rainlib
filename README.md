## 使用说明
  * rainlib是彩虹装修系统的前端渲染包，负责从后端加载数据，在指定的区域渲染数据
  * 通过gallary引入rainlib组件
  * var rain = new Rainlib()
  * rain.setup(id, sessionid) 渲染装修内容
  * id：需要渲染区块的ID编号
  * sessionid：当前用户的登录id
